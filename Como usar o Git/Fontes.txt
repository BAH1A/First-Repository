Alguns comandos:
cd + nome da pasta = entra na pasta pelo terminal do git
git clone + link do repositorio do github = É clonado uma pasta do repositorio do github no arquivos do pc
git status = verifica como está a situação dentro da pasta
git git add . = coloca as modificações como controle de versão local
git commit -m "mensagem de atualização" = adiciona um comentario às modificações
git push origin main = envia todos os commit para a nuvem
